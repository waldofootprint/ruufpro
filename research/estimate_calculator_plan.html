RuufPro Estimate Calculator — Final Build Plan

V2 (Google Solar API) + Full Accuracy Stack • Updated 2026-03-27

What this is: The estimate widget is RuufPro's core paid product ($99/mo). Homeowners enter their address, get an instant satellite-measured price range, and become leads. We use every data point from Google's Solar API — roof area, pitch, segment complexity — plus geometric inference, weather adjustment, and a calibration loop to match or exceed Roofle's accuracy at a fraction of the cost.

How It Works

Homeowner Flow (V2 — Happy Path, ~80% of addresses)

  1. Enters address (Google Places autocomplete) + picks material (asphalt / metal / tile)
  2. We call Google Solar API → get roof sqft, pitch, and segment complexity
  3. Two quick refinement questions: stories (1/2/3) and shingle layers (1/2/not sure)
  4. Full calculation with all factors → show price range
  5. "Get Your Exact Quote" → enters name, phone, email → becomes a lead

Homeowner Flow (V1 — Fallback, ~20% of addresses)

If Solar API returns no data:

  1. Show: "We'll need a few more details about your home"
  2. Enter bedrooms, stories, roof type (gable/hip/flat), shingle layers
  3. We estimate sqft from those inputs → show price range
  4. Same lead capture step

Roofer Flow (Dashboard)

  1. Set $/sqft per material with regional pricing suggestions
  2. Set service area (ZIP codes or cities)
  3. Preview widget with their pricing
  4. Copy embed code for external sites
  5. View leads with estimate details attached
  6. Order EagleView detailed reports for serious leads ($25-50)

Calculation Logic

V2 (Solar API — Full Data Utilization)

From Solar API (buildingInsights endpoint):
  roof_area      = wholeRoofStats.areaMeters2 → convert to sqft
  pitch_degrees  = wholeRoofStats.pitchDegrees
  num_segments   = count of roofSegmentStats[]

From homeowner (2 quick questions after Solar lookup):
  stories        = 1 / 2 / 3
  shingle_layers = 1 / 2 / "not sure"

PITCH_MULTIPLIER (labor cost for steep roofs):
  0-18°  (low slope):     1.00
  19-26° (walkable):      1.05-1.10
  27-34° (steep):         1.15-1.25
  35-45° (very steep):    1.30-1.50

WASTE_FACTOR (extra material based on roof complexity):
  2 segments (simple gable):    1.10 (10%)
  3-4 segments:                 1.15 (15%)
  5-6 segments:                 1.18 (18%)
  7+ segments (complex hip):    1.22 (22%)

STORY_FACTOR:  1-story: 1.00 | 2-story: 1.05 | 3-story: 1.10

TEAROFF_COST (per sqft):
  1 layer: $0 | 2 layers: +$1.00-1.50/sqft | "not sure": $0

GEOMETRIC INFERENCE (from segment data):
  Calculate shared edge lengths between adjacent segments:
    ridge_length, hip_length, valley_length, perimeter
  RIDGE_CAP    = ridge_length × $2.50-4.00/ft
  HIP_FLASH    = hip_length × $3.00-5.00/ft
  VALLEY_FLASH = valley_length × $4.00-6.00/ft
  STARTER      = perimeter × $1.50-2.50/ft

PENETRATION ESTIMATE (statistical by roof size):
  <1,500 sqft: $400 | 1,500-2,500: $600 | >2,500: $800

FULL CALCULATION:
  material_sqft  = roof_area × waste_factor
  material_cost  = material_sqft × contractor_rate[material]
  labor_cost     = material_cost × pitch_multiplier × story_factor
  accessory_cost = ridge_cap + hip_flash + valley_flash + starter_strip
  tearoff_cost   = roof_area × tearoff_rate
  penetration    = penetration_cost

  price_low  = material_low + labor_low + accessory_low + tearoff_low + penetration_low
  price_high = material_high + labor_high + accessory_high + tearoff_high + penetration_high

Output: "$9,800 - $14,100 for asphalt shingles"
        "Based on 2,145 sqft roof · 5/12 pitch · satellite-measured"

V1 (Manual Fallback — ~20% of addresses)

BEDROOM_SQFT_MAP = { 1: 750, 2: 1100, 3: 1500, 4: 2000, 5: 2800 }
ROOF_FACTOR      = { gable: 1.3, hip: 1.4, flat: 1.05 }
Default pitch_multiplier = 1.10 | Default waste_factor = 1.15

Same formula as V2 but with estimated roof_sqft from bedroom lookup.

What We Intentionally Don't Estimate (Requires Inspection)

  • Decking condition / replacement ($2-4/sqft if rotten)
  • Access issues (trees, fences, power lines)
  • Code upgrades (ventilation, ice/water shield)
  • Exact penetration count (we use statistical estimate)

These are handled during the roofer's free inspection — the estimate is the hook.

Google Cloud Setup (Hannah)

  1. Create Google Cloud account (free, $300 credit for new accounts)
  2. Create a new project ("RuufPro")
  3. Enable: Solar API + Places API (New)
  4. Create API keys (restrict each to its respective API)
  5. Add to .env: GOOGLE_SOLAR_API_KEY and GOOGLE_PLACES_API_KEY

Cost: Solar API free tier = 10,000 calls/mo. Places = ~$0.003/session. Total at MVP scale: ~$0-5/mo.
